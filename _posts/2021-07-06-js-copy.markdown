觉得手动copy麻烦的话，写一个小脚本，执行即可copy  
提示用户体验

```javascript
function copy(val) {
  var $temp = $("<input>");
  $("body").append($temp);
  $temp.val(val).select();
  document.execCommand("copy");
  $temp.remove();

  alert("Link copied successfully!");
}
```
